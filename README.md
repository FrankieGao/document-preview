# document-preview
unoconv : creates a preview image of given document ( .doc, .xls, .odt, .txt, .html, ... )

url:  
http://dag.wiee.rs/home-made/unoconv/  
https://wiki.ubuntuusers.de/unoconv/


### installation:  
```
sudo apt-get install unoconv
```


### usage:
start the unoconv listener
```
unoconv -h
```
